# luffy的博客

> A Vue.js project
## 一款基于vue + element-ui的个人博客，目前是纯前端实现，后期将用spring boot搭建后台，MySQL做数据库。
### 目前前端离线版已经完工
### 基本功能
- [x] 首页
- 信息介绍
- 走马灯
- 日历
- 最新博客，最近待办，最新留言
- [x] 博客文章
- 文章列表
- 文章详情
- [x] 待办事项
- 添加、删除
- [x] 关于我
- 导出为PDF
- [x] 留言板
- 留言、评分
